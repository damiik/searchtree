# searchtree
## MERN React-Redux 

### Simple example with:

    * React 16.7.0
    * Redux 4.0.1
    * Express.js server 
    * Webpack scripts
    * Bootsrap
    * MongoDB
    * Babel 7.2.3 (@babel/core 7.2.2) (server and client side)


#DEPLOY:

    yarn build-server
    yarn postinstall
    git add .
    git commit --amend
    git push -f liveServer master
